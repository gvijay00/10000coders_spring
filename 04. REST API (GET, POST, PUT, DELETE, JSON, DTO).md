# ğŸŸ¦ **9. POST API (To Receive JSON Data)**

### What is DTO?

DTO = Data Transfer Object
Used to receive request data and send response.

### Create DTO:

```java
public class StudentDTO {
    private int id;
    private String name;
    private String city;

    // getters and setters
}
```

### Create API:

```java
@PostMapping("/student")
public String saveStudent(@RequestBody StudentDTO dto) {
    return "Student Saved: " + dto.getName();
}
```

### JSON Body (Postman):

```json
{
  "id": 1,
  "name": "Vijay",
  "city": "Hyderabad"
}
```

# ğŸŸ§ **12. ResponseEntity (Beginner Explanation)**

ResponseEntity helps to return:

* Custom status code
* Custom response message

### Example:

```java
@PostMapping("/register")
public ResponseEntity<String> register(@RequestBody StudentDTO dto) {
    return ResponseEntity
            .status(HttpStatus.CREATED)
            .body("Registered: " + dto.getName());
}
```
